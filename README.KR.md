# 🚀 Ultimate Multilingual ATS Resume Builder

[영어 버전으로 보기](./README.md)

**ATS(Applicant Tracking System, 채용 관리 시스템) 친화적인 이력서를 위한 웹 기반 올인원 제작 솔루션입니다.** 
복잡한 문서 도구 없이 브라우저 환경에서 즉각적으로 이력서를 작성, 관리, 출력할 수 있는 고성능 제작 도구입니다.

## 🛠 주요 기술 및 특징

* **상태 기반 UI 렌더링**: 사용자의 입력과 `state` 객체를 실시간 동기화하여, 데이터 변경 시 미리보기 화면을 즉시 재구성합니다.
* **다국어 지원 (i18n)**: 영어와 한국어 UI를 완벽하게 지원하며, 설정 변경 시 즉시 언어 체계가 전환됩니다.
* **로컬 스토리지 데이터 영속성**: 브라우저의 `LocalStorage`를 사용하여 작성 중인 데이터를 자동으로 백업합니다.
* **인쇄 최적화 엔진 (@media print)**: 
    * 시스템 기본 메타데이터를 제거하고 A4 규격에 맞춘 인쇄 환경을 제공합니다.
    * 템플릿별 마진과 폰트 사이즈를 미세 조정하여 완벽한 인쇄물을 보장합니다.
* **유연한 템플릿 시스템**: Modern, Classic(ATS), Executive, Minimal 등 4종의 레이아웃을 즉시 전환할 수 있습니다.

## ✨ 주요 기능

* **실시간 라이브 프리뷰**: 입력 내용이 우측 A4 캔버스에 즉시 반영되는 직관적인 환경.
* **데이터 관리 파이프라인**: JSON 데이터 내보내기/가져오기 기능 및 전체 데이터 초기화.
* **동적 섹션 제어**: 경력, 학력, 프로젝트 등의 항목을 원하는 만큼 유연하게 추가/삭제 가능.
* **프로필 이미지 최적화**: FileReader를 통한 이미지 업로드 및 실시간 반영.

## 📋 사용 방법

1. **입력**: 좌측 에디터 패널에 정보를 입력하세요. 작성 내용은 실시간으로 자동 저장됩니다.
2. **커스터마이징**: 상단 설정바에서 원하는 언어, 테마, 템플릿을 선택하세요.
3. **데이터 보관**: 하단의 **Export JSON** 버튼을 눌러 소중한 데이터를 파일로 백업하세요.
4. **출력**: 우측의 **Download PDF** 또는 **Print** 버튼을 사용하여 제출용 파일을 생성하세요. (인쇄 시 '여백 없음' 설정을 권장합니다.)
